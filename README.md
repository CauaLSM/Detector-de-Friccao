# Friction Detector & UX Insights 🚀

Este projeto visa criar uma ferramenta de análise de comportamento de usuário que identifica pontos de "fricção" em websites. Através de mapas de calor e inteligência artificial multimodal, transformamos dados de navegação em tickets de melhoria de UX acionáveis.

## 👥 Equipe
Projeto desenvolvido por: Caio, Max e Cauã.

## 🛠️ Stack Tecnológica (Proposta)
- **Frontend Tracker:** JavaScript (Vanilla)
- **Dashboard:** React + Tailwind CSS
- **Backend:** Python (FastAPI)
- **IA/Vetorização:** Python + Gemini API / OpenCV
- **Infra/Monitoramento:** Docker, Grafana e Zabbix

## 🏗️ Estrutura do Projeto
- `/tracker`: Script JS para ser inserido nos sites dos clientes.
- `/api`: Servidor backend para recebimento e processamento de eventos.
- `/ai-engine`: Módulo de processamento de imagem e integração com IA.
- `/dashboard`: Interface para visualização de heatmaps e gestão de tickets.
